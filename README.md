# devoir-MySQL-tifosi

## Initialisation

### Création de la base de donnée
La base de données tifosi est créée avec le jeu de caractères utf8mb4 afin d'assurer une compatibilité complète avec les caractères Unicode et les accents.

### Création de l'utilisateur "tifosi"
Un utilisateur spécifique intitulé 'tifosi' est créé avec l'ensemble des privilèges administrateur pour la base de donnée 'tifosi'

Depuis un terminal, positionez vous sur le dossier du projet avec la commande

`cd chemin du projet`

La commande suivante permet de se connecter avec l'utilisateur 'tifosi'

`mysql -u tifosi -p`
